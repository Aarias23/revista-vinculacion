# Sticky Navbar Demo

This page demonstrates the sticky navbar using the production layout, styles, and JavaScript.

Scroll through the sample content to verify persistent visibility, the subtle scroll state, keyboard focus, and the responsive menu.

## Demo Content

The navbar remains at the top while this content scrolls below it. On smaller screens, the navigation collapses into the hamburger menu while preserving the same links and primary gallery action.

### Section One

The component uses the institutional color palette and existing typography to maintain visual consistency across the magazine.

### Section Two

Keyboard users can tab to the brand, menu button, and each navigation link. Pressing Escape closes the mobile menu.

### Section Three

The sticky behavior uses normal document flow, so the navbar does not cover the page content.

### Section Four

Customization options are documented in `docs/STICKY_NAVBAR.md`.
